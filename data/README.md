# Cbarrgs Marketing Agent Worktree

This directory serves as the "brain" for managing Cbarrgs' social media presence and marketing efforts.
It contains a structured file system to keep track of ideas, schedule content, and monitor growth.

## Structure

- `content_ideas.md`: A living document of content concepts for Instagram Reels and TikToks.
- `schedule.csv`: A posting schedule to maintain consistency.
- `metrics/growth_tracker.md`: Weekly logs of follower counts and engagement metrics.
- `scripts/`: A place to store automation scripts (e.g., node scripts to fetch YouTube API stats).
